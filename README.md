# Dashboard interactivo para el análisis de partidos de la Liga MX desde 2016 hasta 2024.
#  Análisis Liga MX (2016–2024) – Power BI

##  Objetivo
Analizar el comportamiento ofensivo de la Liga MX, comparando goles por temporada,
local vs visitante y rendimiento de equipos.

##  Dataset
- Fuente: Dataset histórico de partidos Liga MX
- Periodo: 2016 – 2024
- Registros: Partidos oficiales de liga

##  Métricas principales
- Total de partidos
- Total de goles
- Promedio de goles por partido
- Goles local vs visitante
- Ranking de equipos por victorias

##  Herramientas
- Power BI Desktop
- DAX
- Power Query
- GitHub

##  Cómo usar el proyecto
1. Descarga el repositorio
2. Abre el archivo `.pbix` en Power BI Desktop
3. Asegúrate de que el CSV esté en la carpeta `/data`
